### 1. Vectors & Data Manipulation (`vectorStuff.cpp`)
Explored the `std::vector` container and different ways to iterate through data:
* **Classic For-Loops:** Using indices `for(int i = 0; i < vec.size(); ++i)`.
* **Range-based For-Loops:** Modern C++ syntax `for(int a : vec)`.
* **Auto Keyword:** Utilizing `auto` for cleaner, type-safe iterations.

### 2. Object-Oriented Programming (`classStuff.cpp`)
Building the foundation of OOP with a focus on:
* **Constructors:** Implementing both default and parameterized constructors.
* **Encapsulation:** Managing access levels using `public` and `private` members.
* **Const Correctness:** Learning where, when, and why to use `const` to write safer, read-only code.

### 3. File I/O & Parsing
Practical application of data handling:
* **`std::ifstream`:** Opening and reading from `.txt` files.
* **Data Parsing:** Tokenizing strings from a file (like a student list) and converting them into class objects.
